SOFTWARE

In this section I will go over the software that you will use to convert your current and future physical media (e.g. DVDs) into a digital format for consumption via Plex.

MakeMKV

This software will take your DVDs and Blue-Rays and convert them to a digital format onto your computer.

  • For Windows download click here
  • For Mac download click here

Install the software on your computer. The steps are the same as when you installed the Plex Media Server. You can refer to those intructions here. Once you have installed the program, you may follow these instructions to convert your media. Note that these instructions are meant to be referenced once you are ready to begin creating your library. If you are following the guide for the first time, you may continue to install the next piece of software and then continue on to the Subcriptions section.

  1. Run the program MakeMKV
  2. Click on the "View" menu options then go to "Preferences"
  3. Slect the "Semi-Auto" option
  4. Click on the folder icon. This will bring up a window. Here you will select the "Plex Media" folder. Now, in the next step, you will always be brought to the "Plex Media" folder as your base option. Click "OK"
  5. Click "OK" again
  6. Insert your media into your disc drive in your computer
  7. Click on the big "DVD" button in the middle of the program window to begin conversion
  8. After a moment, a window will appear. Choose the titles you would like to save
    • The title with the most chapters is the main movie. Everything else is trailers etc. Expand the title with the most chapters and select which audio versions and subtitles you want

  9. For your Output folder, select the appropriate sub-folder (i.e. Movies or TV Shows)
  10. Click on the "Save MKV" button to begin your process
    • If a screen pops up asking to create a folder if it does not exist, click yes

  11. Wait for the process to finish
  12. Once it has finsihed converting, go to the folder where you sent the converted file and rename it according to the guidelines set previously. You can reference them here


Now that you have successfully converted your media, you may find that the file size is too large to store many of them if you do not have a lot of storage space. That's where this next piece of software comes in. Handbrake will allow you to convert the file you just created into a more compressed version.

Handbrake

  • For Windows download click here
  • For Mac download click here
  1. Install the software
  2. Run the software
  3. Select "Presets" > Presets > High Profile
  4. Select "Source" then choose "File"
  5. Browse to where you saved your converted file from MakeMKV. Select it then click "Open"
  6. For your Destination, click browse. The same location should appear
  7. For the File Name, give it the same name as previously and click "Save"
  8. Click "Start"
  9. Once process has finished, you may close the program and delete the .mkv file as you have now created a more compressed .mp4 version
